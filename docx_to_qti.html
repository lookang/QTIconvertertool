How to format your document

  • Start each question with a number (e.g. 1 What is...)
  • Images with text overlay should be "cleaned up" by replacing them with screenshots.
  • MCQ options should be labelled A. / B. / C. / D. or A) / B) / C) / D)
  • Structured questions will be extracted as a whole (including parts/subparts).
  • Indents (e.g., a), (i), etc) should not be used; replace them with plain text instead.
  • Append answers at the bottom and begin the section with the capitalised word ANSWERS.
  • Answers for parts/subparts: the question number appears only on the first part — see sample doc Q4.
  • Mark scheme: upload the matching MS file above to automatically populate correctResponse in each QTI item.
